# TV_Slideshow
Простое веб-приложение для создания слайд-шоу на телевизорах с Android TV или любом браузере.

## ✨ Возможности

- 🔵 **Google Drive** — загрузка по ссылкам на изображения
- 🔗 **Прямые ссылки** — любые URL изображений
- 📱 **Локальные файлы** — выбор файлов или папок с устройства
- ⚙️ **Настройки** — интервал от 2 сек до 5 мин
- 🎨 **Эффекты** — fade, slide, zoom
- 🎮 **Управление** — клавиатура, пульт ДУ, тач

## 🚀 Быстрый старт

1. Скачайте файл `index.html`
2. Откройте его в браузере на Android TV
3. Загрузите изображения любым способом
4. Начните слайд-шоу

## 📥 Способы загрузки изображений

### Google Drive
1. Откройте папку в Google Drive
2. Выделите все фото (Ctrl+A)
3. Нажмите «Поделиться» → «Копировать ссылки»
4. Вставьте ссылки в приложение

### Локальные файлы
- Выберите отдельные изображения или целую папку

### Прямые ссылки
- Вставьте любые URL изображений через запятую или с новой строки

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| ← → | Листать изображения |
| Пробел | Пауза / Воспроизведение |
| ↑ | Изменить интервал |
| ↓ | Случайный порядок |


## 📱 Поддерживаемые платформы

- Android TV
- Google TV
- Любой современный браузер
- Планшеты и смартфоны
